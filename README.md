# Advanced CSS3 & Responsive Architecture

## Project Overview

This project demonstrates advanced CSS3 concepts including:

- CSS Grid Layout
- Flexbox Layout
- Responsive Design
- Media Queries
- CSS Variables
- Light/Dark Theme

## Features

### CSS Grid

Used for page-level layouts.

### Flexbox

Used for project card alignment.

### Responsive Design

Supports:

- Mobile
- Tablet
- Desktop

### Theme Switching

Dark mode and light mode using CSS Variables.

## Technologies

- HTML5
- CSS3
- JavaScript
